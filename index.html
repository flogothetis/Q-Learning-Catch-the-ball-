Description of the Game

Abstract

Reinforcement Learning includes techniques used towards "Online Learning". Q-Learning Algorithm is well-known for its effectiveness and robustness. Keeping that in mind, we built a Q-Learner Agent who his target is to learn all the correct movements that are needed to win a game. Specifically, the game is called "Catch the ball". The game environment contains a falling ball and a moving object. The target is to learn the way to move that object in order to avoid missing the ball. The ball is fallen down randomly between eight different poses. A Q-Learner Algorithm was developed to simulate the best-player of the game .
