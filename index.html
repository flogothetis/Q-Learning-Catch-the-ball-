Description of the Game

Abstract

The target is to built a Q-Learner Agent to catch the ball. The canvas contains a cicle as a ball and a rectagular as a catcher of the ball . The ball is fallen down randomly between eight different poses.A Q-Learner Algorithm is built to simulate the Best-player of these Game .
